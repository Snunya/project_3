# Dependency Visualizer

Инструмент визуализации графа зависимостей для Maven пакетов.

## Этап 1: Минимальный прототип

Реализовано:
- Парсинг аргументов командной строки
- Валидация входных параметров
- Вывод параметров в формате ключ-значение
- Обработка ошибок

## Использование

```bash
python dependency_visualizer.py -p "groupId:artifactId" -v "version" [options]
